<h1 align="center">NiKS</h1>

<p align="center">
  Konfigurasi NixOS berbasis flakes dengan Hyprland dan Home Manager.
</p>

---

### 🔧 Fitur
- Dukungan Flakes modern
- Hyprland sebagai WM utama
- Konfigurasi modular dan mudah dikustomisasi
- Home Manager untuk manajemen dotfiles

### 📁 Struktur
- `configuration.nix` – Konfigurasi utama sistem
- `home/home.nix` – Konfigurasi pengguna (Home Manager)

---

> 💡 Sesuaikan isi konfigurasi sesuai preferensimu.
